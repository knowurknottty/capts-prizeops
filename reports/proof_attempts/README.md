# Proof Attempts Log

Each directory contains a complete run of the CAPT solving pipeline:

- `manifest.json` — Run configuration and CAPT state
- `ledger.json` — Verified claims
- `preprint.md` — arXiv-style paper template
- `formal/` — Lean/Isabelle formalization files
- `search_results/` — Raw computational outputs
- `arena_results/` — Arena comparison data (if applicable)
